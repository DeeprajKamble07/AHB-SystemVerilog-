# AHB-SystemVerilog-

This project focuses on the functional verification of the **AMBA Advanced High-performance Bus (AHB)** protocol for **INCR4 burst transactions** using SystemVerilog. The verification environment validates correct behavior of the AHB master during 4-beat incrementing burst transfers, ensuring proper generation of control signals such as HADDR, HTRANS, HWRITE, HBURST, HSIZE, and handling of HREADY and HRESP. A self-checking testbench is developed using SystemVerilog constructs, including stimulus generation, driver, monitor, and scoreboard, to verify address incrementing, data consistency, and protocol compliance across all burst beats. Simulation results confirm correct sequential address progression and data transfer for INCR4 bursts without protocol violations, demonstrating reliable AHB master functionality for high-performance SoC designs.
